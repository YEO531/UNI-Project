# Uni Project Code Archive

This repository contains code files collected from university project folders across:

- Year 1
- Year 2
- YEAR 3

The sync was prepared from local folders:

```text
C:\Users\YEO\OneDrive\Desktop\Degree\Year 1
C:\Users\YEO\OneDrive\Desktop\Degree\Year 2
C:\Users\YEO\OneDrive\Desktop\Degree\YEAR 3
```

## What Is Included

The repository keeps source-code and project configuration files, such as:

- Python, Java, C/C++, C#, PHP, JavaScript, HTML, CSS, SQL
- Jupyter notebooks
- project/config files such as `.sln`, `.csproj`, `.json`, `.xml`, `.yml`, `.md`, and `requirements.txt`

## What Is Excluded

The sync intentionally excludes non-code and generated files, such as:

- ZIP/RAR archives
- PDFs, Word documents, PowerPoint files, Excel files
- videos, images, executables, databases
- build output folders such as `bin`, `obj`, `dist`, `build`, `target`
- dependency folders such as `node_modules`, `.venv`, `venv`

## Sync Manifest

`sync_manifest.json` records the source folders, copied file count, skipped file count, and copied file paths for audit/reference.
